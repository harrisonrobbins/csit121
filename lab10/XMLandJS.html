What is XML?

XML stands for eXtensible Markup Language whose purpose is to store and transport data while being able to be read by both humans and machines.

How is XML used?

XML doesn't carry information about how to be displayed, and thus is used to separate the data from the presentation.

What does XML look like? (provide an example)

XML code may look somewhat similar to HTML, but not quite exactly the same. XML elements must be properly nested, so for example, an italics tag inside a bold tag must be formatted like this:

<b><i>This text contains text.</i></b>

You can also write basically whatever for a tag name, whereas in HTML it's all pre-defined.

<parent>
   <child>
      <subchild>This is a sentence.</subchild>
   </child>
</parent>

What is JavaScript?

JavaScript is the number one most popular programming language in the world and is (supposedly) easy to learn.

How is JavaScript related to HTML and CSS?

JavaScript serves distinctly different purposes from HTML and CSS, but they all work together to create a functioning web page.

What are some common JavaScript elements, properties, methods, and functions? (provide at least 1 example of each)

JS doesn't have elements per sé unless you refer to JavaScript HTML DOM Elements. Otherwise, you'd refer to them as objects, probably. One example of a JS object is something like arrays, though anything other than primitives can be or are objects.
JS properties can be written as either objectName.propertyName, objectName["propertyName"], or objectname[expressionName]. One example is the code person.age = "20";.
One example of a JS method is the different ways to get and set dates. For example, the method setTime() sets the number of milliseconds that have passed since January 1st, 1970.
JS functions are declared using the function keyword. One example of a JS function is the following code:

function functionName(parameterA, parameterB) {
   *insert code here*
}
